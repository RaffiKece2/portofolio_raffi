Angry Birds

Di pulau para burung, kawanan babi hijau mencuri telur-telur milik para Angry Birds. Karena marah, para burung memutuskan untuk menyerang markas para babi untuk merebut kembali telur mereka. Setiap level menggambarkan perjalanan burung-burung ini menghancurkan benteng dan struktur yang dibuat para babi menggunakan ketapel. Semakin jauh, babi makin cerdik membangun pertahanan, hingga akhirnya burung berhasil mengalahkan raja para babi dan menyelamatkan telur-telur mereka. Meskipun sederhana, inti cerita Angry Birds adalah perjuangan para burung merebut kembali telur yang dicuri dan menghentikan para babi dari rencana jahat mereka.
